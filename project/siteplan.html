Cherokee Run Club Site Plan

Ryan Worsham

Site Name

Cherokee Run Club

Cherokee Run Club is a running club based out of Cherokee County Georgia

Site Purpose

The purpose of the site will be to provide information about the club. The site will also include information about upcoming events in the community that members and those interested can learn more about. A form for requesting membership will be included.

Scenarios

Where can I find information about club meetups?

What are some good running shoes for beginners?
